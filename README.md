# PHP-VIM-CONFIG

### WHAT IS IT 
**VIM配置文件，将VIM配置成PHP开发的IDE**

### HOW TO USE
**git clone 项目地址 $HOME**

### FORM
* 基本配置（语法高亮，设置Tab空格数，....）
* Vundle插件管理工具
* NerdTree树形目录插件
* phpcomplete && neocomplcache代码补全（设置了Tab功能）
* PHPDoc文档注释插件
* Ctags函数定义跳转插件
* TagsList函数列表生成插件
* Cscope函数使用跳转插件
